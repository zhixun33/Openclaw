What is OpenClaw

龙虾是什么

一句话：运行在你电脑上的 AI 员工

💡 本质

ChatGPT 是云端的对话工具，它能帮你思考，但不能帮你行动。OpenClaw 是本地运行的 AI Agent，它不只是回答问题，而是真正执行任务——打开网页、读写文件、发送消息、操作飞书、定时触发工作流。你的数据完全在本地，不上传任何内容。

记忆能力对话结束即忘持久记忆，越用越了解你

⚠️ Token 消耗提醒

龙虾每个任务会触发 5～10 次 API 调用，每次都会重发完整对话上下文。长会话积累大量历史后 token 消耗会快速上升。养成习惯：复杂任务结束后发 /new 开启新会话，避免上下文无限积累。

FAQ

常见问题

开始前最常被问到的问题

OpenClaw 和 Dify、Coze 有什么区别？▾
Dify 和 Coze 是云端 Agent 平台，你的数据和工作流在他们服务器上。OpenClaw 是本地运行，数据不离开你的电脑，可以直接访问本地文件和系统，且完全免费。代价是需要自己部署，但这份指南帮你解决了这个问题。
关掉电脑龙虾就停了吗？▾
默认是的。但完成部署指南第 09 步「持续运行」后，龙虾会注册为系统 LaunchAgent，Mac 插电且屏幕关闭时依然在后台运行，飞书消息随时可以触达它。完全关机才会停。
数据安全吗？会不会被看到？▾
OpenClaw 本身完全本地运行，不上传你的文件和对话内容。但你接入的 AI 大脑（DeepSeek、Qwen 等）是云端 API，你发给它的消息会传到对应厂商服务器。如果有隐私敏感数据，使用本地 Ollama 模型可以做到完全不出本机。
不懂代码能用好吗？▾
部署需要跟着本指南在终端执行几十条命令，这本身不需要懂代码，只需要复制粘贴。部署完成后，日常使用完全通过自然语言在飞书对话，不需要任何编程知识。这份指南正是为不懂技术的用户写的，每步都有说明和成功示例。
同时用飞书和网页两个入口会冲突吗？▾
同一个对话不会冲突，两边都能看到记录和互相同步。两个不同对话同时执行不同任务也没问题。唯一需要注意的是：两边同时操作同一个文件时可能产生冲突，执行重要任务时保持一个入口操作即可。
iPhone 上能用吗？▾
间接可以。接入飞书后，手机飞书 App 就是控制龙虾的入口，随时随地发消息驱动它工作。2026.2.19 版本还支持了 Apple Watch，可以在手腕上收到龙虾主动推送的消息。

Step 01

代理 + Homebrew

地基，没有它后面装不了任何东西

⚡ 每次打开终端先执行

国内直连 GitHub 经常超时。ClashX Pro 默认端口 7890，每次新开终端窗口先执行这两条，是所有步骤能跑通的前提。

设置代理 + 安装 Homebrew
# 每次开终端先执行 ~ % export https_proxy=http://127.0.0.1:7890 ~ % export http_proxy=http://127.0.0.1:7890 # 验证代理（看到 HTTP/2 200 就成功） ~ % curl -I https://github.com # 安装 Homebrew（密码输入时不显示字符，盲打回车） ~ % /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" # M1 必须执行这两条 ~ % echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile ~ % eval "$(/opt/homebrew/bin/brew shellenv)" ~ % brew --version Homebrew 4.4.x ← 看到版本号就成功
🚨
curl: (56) Recv failure: Operation timed out
GitHub 连接超时，安装脚本下载失败
✅ 解决方案

确认终端代理已开启（执行上面两条 export 命令），然后重新运行安装命令。

Step 04

获取 API Key

推荐 DeepSeek，注册即用，无需特殊网络

⚠️ 安全提醒

Key 只在创建时显示一次，立刻存到备忘录。不要把 Key 粘贴在任何 AI 对话里。 暴露了立刻去平台删掉重建。

  • 1
    浏览器打开 https://platform.deepseek.com，手机号注册
  • 2
    登录后点左侧 「API Keys」→「+ 创建 API Key」
  • 3
    复制 sk- 开头的 Key，存到备忘录
🚨
百炼 Coding Plan Key 一直报 401
HTTP 401: invalid access token or token expired
✅ 原因 + 解法

sk-sp- 开头的 Coding Plan Key 只支持代码补全，不支持对话。加上 OAuth 失败会留下残留配置。最简单：直接换 DeepSeek Key。

🚨
Qwen OAuth 显示「账号未注册」
✅ 解法

阿里云账号和 chat.qwen.ai 是两套体系。Ctrl+C 退出向导，重新运行 openclaw onboard，选 Other/Custom，手动填写 baseUrl 和 Key。

Step 10

SOUL + USER

让龙虾认识你，从失忆状态变成专属助手

💡 设计原则

不写身份标签（「我是 XX 专家」会限制联想），只写思维方法论。USER.md 放不变的生活事实，SOUL.md 放行为操作系统。两个文件控制在 300 字以内，越精简越不傻。

打开文件
~ % open ~/.openclaw/workspace/USER.md ~ % mkdir -p ~/.openclaw/agents/default && open ~/.openclaw/agents/default/SOUL.md

USER.md 参考模板

👤USER.md — 只写不变的事实
